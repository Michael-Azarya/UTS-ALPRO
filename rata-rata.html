Menghitung Rata-Rata

Pada bagian ini, anda akan melihat bagaimana algoritma dalam menghitung rata-rata dari beberapa bilangan.

Andrew Steine Michael Azarya

Deskripsi

Menghitung rata rata dapat dilakukan dengan menambahkan nilai dari variabel lalu membagi total nilai variabel tersebut berdasarkan jumlah variabel yang ada

Flowchart

Pseudocode

1. mulai
2. fungsi Hitung rata-rata(array):
3. jika panjang(array) == 0: kembalikan "Array kosong"
4. total = 0 untuk setiap bilangan dalam array:total += bilangan
5. ratarata = total / panjang(array)
6. kembalikan rata-rata
7. selesai

code in c++

#include iostream>
using namespace std;

	int main() {
		int n;
		double number, sum = 0, average;
							
		// Akan memunculkan berapa bilangan yang ingin dihitung rata ratanya
		cout << "Masukkan jumlah bilangan: ";
		cin >> n;
							
		for(int i = 1; i <= n; i++) {
		cout << "Masukkan bilangan ke-" << i << ": ";
		cin >> number;
		sum += number;
	        	}
							
		average = sum / n;
		cout << "Rata-rata dari " << n << " bilangan adalah: " << average;
							
		return 0;

		}
